# Shadow Hook Panel v9

Features
- Responsive phone and desktop layout
- Multi-user accounts
- Private webhook and scheduler per user
- Multiple image uploads
- Multiple video uploads
- Mixed image + video sending in one message
- Discord OAuth connect button
- Discord avatar shown in the top-right BH area after connect

Run
npm install
npm start

Important for Discord account connect
You must set these env values:
- DISCORD_CLIENT_ID
- DISCORD_CLIENT_SECRET
- DISCORD_REDIRECT_URI
- BASE_URL
- SESSION_SECRET
